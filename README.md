# EyeCare Pro

Professional Ocular Integrity & Vision Diagnostic Suite.

## Features
- **Visual Acuity Lab**: Digital Snellen Chart for quick vision assessment.
- **Color Matrix**: Clinical Ishihara plates for color vision diagnostics.
- **Astigmatism Clock**: Interactive meridian detection.
- **Contrast Lab**: Multi-level sensitivity testing.
- **Logicreminders**: Automated 20-20-20 rest protocols with live countdowns.
- **PWA Support**: Install as a native application on mobile and desktop.
